# Moody Foody

Moody Foody is a simple website project for a fictional restaurant where users can explore food categories, view available foods, and contact the restaurant.
`Note:` This project is an enhanced version of the food ordering website project tutorial by Vijay Thapa on Youtube.

## Live Demo
Check out the live demo of the project: [Moody Foody](http://moody-foody.zya.me/)
